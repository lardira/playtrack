# playtrack frontend

playtrack frontend written in sveltekit

## Quick Start

### Prerequisites

- node v(?)

### Installation

1. Clone the repository, then
```bash
cd ./web-app
```

2. Install dependencies and run
```bash
npm install

# default http://localhost:5173
npm run dev
```

#### Build

```bash
npm run build
```

built files will be in `build/`.

```bash
npm run preview
```

#### Misc.

- **check types:** `npm run check`
- **tests:** `npm run test`
